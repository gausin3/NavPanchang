with NavPanchang over a network be able to obtain the complete corresponding

Source code for NavPanchang is available at:
    https://github.com/<placeholder>/navpanchang

To rebuild NavPanchang from source:
 1. Clone the repository.
 2. Ensure Android Studio Koala+ / JDK 17 / Android Gradle Plugin 8.5+.
 3. Place your own signing keystore at `keystore/upload-key.jks` (or use the debug
    keystore for development builds).
 4. Run `./gradlew :app:assembleRelease` from the project root.

The Swiss Ephemeris JAR under `app/libs/` is AGPL v3 itself (dual-licensed with a
commercial option from Astrodienst AG). We use the AGPL option; see
